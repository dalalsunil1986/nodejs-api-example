# nodejs-api-example

Example project for API creation using NodeJS

Tech stack

* sqlite3
* nodejs (see dependencies in package.js), but mainly uses express to serve content

# Support API end-point

Supported end-point is as follows

* `/listUsers`  
   `GET` method
